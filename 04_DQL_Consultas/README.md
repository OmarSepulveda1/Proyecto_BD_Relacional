# DQL — Consultas (SELECT) y reportes

Esta carpeta agrupa ejemplos de consultas SELECT y consultas de reporte pensadas para extraer información del modelo de datos del proyecto.

Archivos principales

- `select_queries.sql` — Consultas de ejemplo: joins, agregaciones (SUM, COUNT), filtros, subconsultas y consultas de reporte.

Cómo usar estas consultas

1. Asegúrate de tener la base de datos creada y con datos (usa `02_DDL_Tablas/create_database.sql` y `03_DML_Manipulacion/insert_data.sql`).
2. Abre `select_queries.sql` en tu cliente SQL (MySQL Workbench, DBeaver, etc.) y ejecuta las secciones que quieras probar.
3. Ajusta los filtros (WHERE) y parámetros para tus datos reales.
